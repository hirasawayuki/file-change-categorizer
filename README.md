# File Change Categorizer
## Overview

File Change Categorizer is a Chrome extension that enhances your GitHub experience by automatically categorizing and labeling files in commit histories and pull requests based on predefined rules.

## Development Setup

To set up your development environment for working on File Change Categorizer, follow these steps:

### Prerequisites

Make sure you have [pnpm](https://pnpm.io/) installed. If you don't have it installed, you can install it via npm:

```sh
npm install -g pnpm
```

### Install Dependencies
Navigate to the project directory and install the necessary dependencies:

```sh
pnpm i
```

### Start Development Server

```sh
pnpm run dev
```

### Building for Production

```sh
pnpm run build
```
